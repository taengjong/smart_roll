학원 출석 어플을 개발할 예정이다.

IOS, Android 앱을 통해 사용자가 해당 서비스를 사용할 수 있음
관리자는 어플과 웹 페이지를 통해서 접근을 할 수 있다.

어플과 웹페이지에서 접근하는 데이터베이스가 같은 곳을 보아야 스케줄 관련 관리가 가능할 것으로 생각함.

개발이 필요한 기능들을 아래에 명시한다.
- 개인 유저에게 할당된 수업에 대한 출석 기능
	- 내가 어느 날에 수업이 잡혀있는지에 대한 수업 조회 화면
- 관리자는 특정 유저에게 수업을 할당할 수 있도록 해야함
	- 관리자 페이지를 통해 각 유저 별 수업 등록 기능
	- 개인 유저가 휴대폰을 가져오지 않는 경우를 대비해 관리자가 관리자 페이지에서 수업을 대신 등록할 수 있는 기능
- 여러 사용권에 대한 앱 내 결제 기능
	- 취미 보컬, 입시 보컬, 취미 기타, 입시 기타 와 같은 수강권이 존재
	- 각 수강권은 미리 구입을 하고 4주 간 수업 참석 가능
- 개인 유저는 어떤 수업에 언제 출석을 했는지, 그리고 수업 결제를 어떻게 했는지에 대한 기록 조회 기능
- 개인 유저는 몇 번 수업을 수강했는지에 대한 통계 화면 필요
- 등록된 수강권에 대해서 스케줄링된 특정 날의 수업 취소 혹은 수업 보강 요청 기능
